## Command 8: touch
- Used to create a new empty file
- Helpful for testing and practice
- Example:
  touch test.txt

## Command 9: cat
- Used to view file content
- Commonly used in Linux
- Example:
  cat test.txt

## Command 10: echo
- Used to write text into a file
- Very useful for logs and notes
- Example:
  echo "Linux practice" > test.txt

## Command 11: nano
- Terminal based text editor
- Easy for beginners
- Example:
  nano test.txt

## Command 12: cp
- Used to copy files
- Keeps original file safe
- Example:
  cp test.txt test_copy.txt

## Command 13: mv
- Used to rename or move files
- Works for both tasks
- Example:
  mv test_copy.txt new_test.txt

## Command 14: rm
- Used to delete files
- Use carefully in Linux
- Example:
  rm new_test.txt

## Command 15: clear
- Clears terminal screen
- Improves readability
- Example:
  clear

## Command 16: history
- Shows command history
- Helpful in investigations
- Example:
  history

## Command 17: whoami
- Shows current logged-in user
- Useful in access control checks
- Example:
  whoami

## Command 18: uname -a
- Displays system and kernel info
- Important for system analysis
- Example:
  uname -a

## Command 19: df -h
- Shows disk space usage
- Helps monitor storage
- Example:
  df -h

## Command 20: free -h
- Shows RAM usage
- Used for system monitoring
- Example:
  free -h

## Command 21: ps
- Displays running processes
- Helps identify active programs
- Example:
  ps

## Command 22: ps aux
- Shows all system processes
- Used in SOC analysis
- Example:
  ps aux

## Command 23: top
- Shows live system resource usage
- Helps detect suspicious activity
- Example:
  top
