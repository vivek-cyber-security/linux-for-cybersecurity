## Commands Practiced

### 1. pwd
This command shows the current working directory.

### 2. ls
This command lists files and folders
present in the current directory.

### 3. whoami
This command displays the username
of the currently logged-in user.

### 4. date
This command shows the current system
date and time.

### 5. uname -a
This command displays detailed system
information such as kernel version
and system architecture.

## Why These Commands Are Important in Cybersecurity
Before analyzing a system, a cybersecurity
professional must know:
- Where they are working (directory)
- Which user is logged in
- What system and kernel is running

These commands help in system identification
and environment verification.
